# H5 TODO 管理系统产品策划案

## 1. 项目概述

本项目旨在开发一个完整的 H5 TODO 管理系统，提供移动端优先的用户体验。系统包含日历视图、快速添加任务、事件编辑、数据持久化等核心功能，采用响应式设计，确保在主流移动浏览器上的兼容性。

## 2. 功能模块划分

### 2.1 日历视图模块 (Calendar View) [已完成]

- **月视图/周视图切换**：用户可根据需要切换查看模式。
- **每日事件标识**：在日历日期下方显示小圆点，标识当天待办事项数量。
- **日期详情**：点击具体日期，下方列表展示该日期的所有 TODO 事项。
- **滑动切换**：支持左右切换月份或周。
- **今天按钮**：快速回到当前日期。

### 2.2 快速添加模块 (Quick Add) [已完成]

- **输入入口**：首页底部放置快速输入框。
- **交互**：
  - 点击输入框唤起键盘。
  - 输入内容后按回车键（或确认键）快速提交。
  - 默认添加到当前选中的日期。
  - 提交成功后触发平滑过渡动画。

### 2.3 事件编辑模块 (Event Edit) [已完成]

- **基础信息**：
  - 标题（必填，大字号无边框设计）
  - 内容/备注（选填，气泡式设计）
- **时间设置**：
  - 开始时间、结束时间（支持精确到分钟）
  - 全天事件开关
- **高级设置**：
  - **重复周期**：支持不重复、每天、每周、每月、每年。
  - **子任务清单**：支持添加/删除/完成子任务，带进度显示。
  - **提醒时间**：支持无提醒至提前 1 天。
  - **优先级**：高、中、低（颜色区分）。
  - **分类标签**：支持选择现有分类或新建分类（颜色标识）。
  - **主题关联**：支持关联自定义主题。

### 2.4 数据存储模块 (Data Storage) [已完成]

- **本地存储**：使用 `localStorage` 进行数据持久化。
- **状态管理**：Zustand Store 管理 Tasks, Categories, Themes, SortOptions。

### 2.5 响应式布局与动画 [已完成]

- **移动端优先**：Tailwind CSS 适配。
- **动画系统**：集成 Framer Motion，实现模态框进出、列表排序、页面切换等 60fps 流畅动画。

### 2.6 分类与主题管理 [新增/已完成]

- **主题管理**：支持自定义主题色、关联优先级。
- **分类管理**：支持创建、删除分类，自定义分类颜色。

## 3. 用户流程设计 (User Flow)

1.  **查看任务**：
    首页 -> 日历视图 -> 点击日期 -> 任务列表（支持排序/过滤） -> 点击任务卡片 -> 任务编辑页。

2.  **快速添加**：
    首页 -> 底部输入框 -> 输入内容 -> 回车 -> 列表自动刷新并显示新任务。

3.  **管理分类/主题**：
    首页顶部 -> 点击文件夹/调色盘图标 -> 打开管理弹窗 -> 新增/删除 -> 关闭。

## 4. 技术选型方案

- **核心框架**：React 18
- **构建工具**：Vite
- **语言**：TypeScript
- **样式库**：Tailwind CSS
- **图标库**：Lucide React
- **动画库**：Framer Motion
- **日期处理**：date-fns
- **状态管理**：Zustand
- **测试框架**：Vitest
